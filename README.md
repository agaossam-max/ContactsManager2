

Projet Java de gestion de contacts.

Ce projet permet de créer, stocker et rechercher des contacts
avec leur nom et numéro de téléphone.

ainsi nous avons:
- Contact.java : pour creer la classe Contact
- ContactsManager.java : elle permet de gerer la liste de contacts
- Main.java : elle démarre le programme

 pour l' exécuter:
1. Ouvrir le projet dans IntelliJ IDEA
2. Configurer le JDK 24
3. Lancer Main.java
